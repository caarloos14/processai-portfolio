# ProcessAI Studio Portfolio

Repositorio con demos y prototipos de automatización inteligente, análisis de datos e inteligencia artificial aplicada a procesos de negocio.

## Proyectos

### 1. Review Analyzer

Análisis automático de reseñas de Google Maps usando Apify, Gemini y Python.

Funcionalidades principales:

- extracción automática de reseñas desde Google Maps;
- limpieza y normalización de datos;
- análisis inteligente con Gemini;
- generación automática de informe HTML;
- detección de puntos fuertes, quejas recurrentes y oportunidades de mejora.
